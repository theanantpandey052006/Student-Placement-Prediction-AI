# 🎓 Student Placement Prediction AI
This project is based on machine learning and data search algorithm

## 📌 Project Overview

This project uses Machine Learning to predict whether a student will get placed based on academic and skill-based factors.

The model analyzes:

- CGPA
- Skills Level
- Internship Experience

and predicts:

- 1 = Placed
- 0 = Not Placed


## 🚀 Technologies Used

- Python
- Pandas
- Scikit-learn
- NumPy
- Machine Learning (Logistic Regression)

Run the project:

## 📊 Sample Output

Prediction: [1]

Where:

1 = Student will get placed  
0 = Student will not get placed  


## 🧠 Machine Learning Model

This project uses Logistic Regression Algorithm.

Steps used:

1. Create dataset  
2. Train model  
3. Test model  
4. Predict placement
